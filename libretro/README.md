## Install

```
~ $ git clone https://github.com/sdsddsd1/kiss-games
~ $ export KISS_PATH=$KISS_PATH:$HOME/kiss-games/equipment
~ $ export KISS_PATH=$KISS_PATH:$HOME/kiss-games/libretro
~ $ kiss b retroarch
~ $ kiss i retroarch
```
